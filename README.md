# 🌾 CropAI Dashboard

A modern **Machine Learning-powered Crop Recommendation System** built with **Streamlit**, **Scikit-Learn**, **XGBoost**, and **Plotly**. The application compares multiple ML algorithms, predicts the most suitable crop based on soil and environmental conditions, and provides detailed model analytics through an interactive dashboard.

---

## 🚀 Features

- 🌱 Crop prediction using Machine Learning
- 🤖 Comparison of 7 ML algorithms
- 📊 Interactive dashboard built with Streamlit
- 📈 Model performance analytics
- 🎯 Accuracy, Precision, Recall & F1 Score comparison
- 📉 ROC Curves & Confusion Matrix
- 🔍 Feature Importance visualization
- 📋 Cross Validation analysis
- 🌍 Interactive data exploration
- 🎨 Modern responsive UI

---

## 🧠 Machine Learning Models

- Random Forest

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/saatwikji/Ankur.git
```

Move into the project directory

```bash
cd Ankur
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
